보조 이메일

회원가입 시 입력하신 보조 e-mail 주소로 e-mail ID 정보를 보내드립니다. ID 찾기를 통한 e-mail ID 정보 제공은 24시간 내 1회만 제공 되며, 24시간 경과 후에 재발송 요청이 가능합니다.

이메일 ID

입력하신 e-mail 주소로 2시간 동안 사용 가능한 임시비밀번호를 보내드립니다. 임시 비밀번호를 받으신 후 2시간 내 로그인 하셔서 비밀번호를 변경 해주시기 바랍니다.
